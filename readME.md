
# Albumize
## An Album Review application


* This application utilizes the following:

1. NODE/EXPRESS SERVER
2. MYSQL DB
3. HEROKU
4. HANDLEBARS



* MySQL Database 
    - Main Table:
        * Users
    - Relational Tables:
        * Top Rated,
        * Most Commented,
        * Latest Review
* GET and POST routes for retrieving and adding new data;
    * GET routes
    * POST routes
