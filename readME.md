
### Requirements


* Use a Node and Express Web Server;


* MySQL Database

        * Main Table:
            - Users
        * Relational Tables:
            - Top Rated,
            - Most Commented,
            - Latest Review
* Have both GET and POST routes for retrieving and adding new data;
    * GET routes
    * POST routes
* Be deployed using Heroku (with Data);
